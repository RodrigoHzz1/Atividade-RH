# 🚀 Nexus ERP - Sistema de Gestão de RH e Clientes

Este projeto consiste num ecossistema Full Stack para a gestão de colaboradores internos (módulo de Recursos Humanos) e controlo comercial de clientes.

---

## 🛠️ Pré-requisitos

Antes de iniciar, certifique-se de que tem instalado na sua máquina:
* **Java JDK 17** ou superior
* **Node.js** (versão 18 ou superior)
* **Maven** (opcional, caso prefira rodar globalmente)

---
